3. matplotlib.cm¶

This module provides a large set of colormaps, functions for registering new colormaps and for getting a colormap by name, and a mixin class for adding color mapping functionality.

3.1. Functions¶

get_cmap([name, lut]) Get a colormap instance, defaulting to rc values if name is None.
register_cmap([name, cmap, data, lut]) Add a colormap to the set recognized by get_cmap().
revcmap(data) Can only handle specification data in dictionary format.

3.2. Classes¶

ScalarMappable([norm, cmap]) This is a mixin class to support scalar data to RGBA mapping.
