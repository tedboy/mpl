14.2.2. matplotlib.text.Annotation¶

class matplotlib.text.Annotation(s, xy, xytext=None, xycoords=u'data', textcoords=None, arrowprops=None, annotation_clip=None, **kwargs)[source]¶

A Text class to make annotating things in the figure, such as Figure, Axes, Rectangle, etc., easier.

Annotate the x, y point xy with text s at x, y location xytext. (If xytext = None, defaults to xy, and if textcoords = None, defaults to xycoords).

axes The Axes instance the artist resides in, or None.
mouseover
stale If the artist is ‘stale’ and needs to be re-drawn for the output to match the internal state of the artist.
xyann
