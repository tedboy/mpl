mpl.tri.TriContourSet¶

class mpl.tri.TriContourSet(ax, *args, **kwargs)[source]¶

Create and store a set of contour lines or filled regions for a triangular grid.

User-callable method: clabel

Useful attributes:
ax:
the axes object in which the contours are drawn
collections:
a silent_list of LineCollections or PolyCollections
levels:
contour levels
layers:
same as levels for line contours; half-way between levels for filled contours. See _process_colors method.
__init__(ax, *args, **kwargs)[source]¶

Draw triangular grid contour lines or filled regions, depending on whether keyword arg ‘filled’ is False (default) or True.

The first argument of the initializer must be an axes object. The remaining arguments and keyword arguments are described in TriContourSet.tricontour_doc.
