mpl.image.pcolor()¶

mpl.image.pcolor(x, y, data, rows, cols, bounds)¶

Generate a pseudo-color image from data on a non-uniform grid using nearest neighbour or linear interpolation. bounds = (x_min, x_max, y_min, y_max) interpolation = NEAREST or BILINEAR
