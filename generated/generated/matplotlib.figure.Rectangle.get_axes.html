8.2.12.19. matplotlib.figure.Rectangle.get_axes¶

Rectangle.get_axes()¶

Return the Axes instance the artist resides in, or None.

This has been deprecated in mpl 1.5, please use the axes property. Will be removed in 1.7 or 2.0.
