21.2.14. matplotlib.collections.TriMesh¶

class matplotlib.collections.TriMesh(triangulation, **kwargs)[source]¶

Class for the efficient drawing of a triangular mesh using Gouraud shading.

A triangular mesh is a Triangulation object.

Attributes

axes The Axes instance the artist resides in, or None.
mouseover
stale If the artist is ‘stale’ and needs to be re-drawn for the output to match the internal state of the artist.
