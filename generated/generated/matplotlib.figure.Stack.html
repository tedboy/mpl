8.2.13. matplotlib.figure.Stack¶

Stack.__init__()

class matplotlib.figure.Stack(default=None)[source]¶

Implement a stack where elements can be pushed on and you can move back and forth. But no pop. Should mimic home / back / forward in a browser

Methods

__call__() return the current element, or None
back() move the position back and return the current element
bubble(o) raise o to the top of the stack and return o. o must be
clear() empty the stack
empty()
forward() move the position forward and return the current element
home() push the first element onto the top of the stack
push(o) push object onto stack at current position - all elements
remove(o) remove element o from the stack
