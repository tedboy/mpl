9.2.10. matplotlib.legend.DrawingArea¶

class matplotlib.legend.DrawingArea(width, height, xdescent=0.0, ydescent=0.0, clip=False)[source]¶

The DrawingArea can contain any Artist as a child. The DrawingArea has a fixed width and height. The position of children relative to the parent is fixed. The children can be clipped at the boundaries of the parent.

Attributes

axes The Axes instance the artist resides in, or None.
clip_children If the children of this DrawingArea should be clipped by DrawingArea bounding box.
mouseover
stale If the artist is ‘stale’ and needs to be re-drawn for the output to match the internal state of the artist.
