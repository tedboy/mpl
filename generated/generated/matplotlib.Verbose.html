1.2.2. matplotlib.Verbose¶

class matplotlib.Verbose[source]¶

A class to handle reporting. Set the fileo attribute to any file instance to handle the output. Default is sys.stdout

Methods

ge(level) return true if self.level is >= level
report(s[, level]) print message s to self.fileo if self.level>=level. Return
set_fileo(fname)
set_level(level) set the verbosity to one of the Verbose.levels strings
wrap(fmt, func[, level, always]) return a callable function that wraps func and reports it
