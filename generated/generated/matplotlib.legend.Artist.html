9.2.1. matplotlib.legend.Artist¶

Artist.__init__()

class matplotlib.legend.Artist[source]¶

Abstract base class for someone who renders into a FigureCanvas.

Attributes

axes The Axes instance the artist resides in, or None.
mouseover
stale If the artist is ‘stale’ and needs to be re-drawn for the output to match the internal state of the artist.
