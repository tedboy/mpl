30.2.15. matplotlib.offsetbox.HPacker¶

class matplotlib.offsetbox.HPacker(pad=None, sep=None, width=None, height=None, align=u'baseline', mode=u'fixed', children=None)[source]¶

The HPacker has its children packed horizontally. It automatically adjusts the relative positions of children at draw time.

__init__(pad=None, sep=None, width=None, height=None, align=u'baseline', mode=u'fixed', children=None)[source]¶
Parameters:

pad : float, optional

Boundary pad.

sep : float, optional

Spacing between items.

width : float, optional

height : float, optional

Width and height of the container box, calculated if None.

align : str

Alignment of boxes.

mode : str

Packing mode.

Notes

pad and sep need to given in points and will be scale with the renderer dpi, while width and height need to be in pixels.
